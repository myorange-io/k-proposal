# 리서치 품질 룰셋 (research-quality-rules)

> 외부 자료(시장규모·경쟁사·정책·특허) 검색·수집·검증·인용에 적용되는 단일 진실 공급원.
> `proposal-research` 스킬과 `researcher` 에이전트가 참조한다. 변경 시 두 곳에 영향이 가므로 신중히.

---

## 1. 소스 신뢰도 5단계 위계 (Tier)

| Tier | 등급 | 예시 출처 |
|------|------|----------|
| **T1** | 정부 공인통계 | KOSIS(kosis.kr), 통계청, DART(dart.fss.or.kr), 한국은행 ECOS(ecos.bok.or.kr), 과기정통부 ICT통계포털, 기획재정부 예산서, 공공데이터포털(data.go.kr) |
| **T2** | 산업협회·공공기관 | KOTRA(kotra.or.kr), NIPA(nipa.kr), KISDI(kisdi.re.kr), KIET(kiet.re.kr), SW산업협회, 한국지능정보사회진흥원, 1차 산업 백서 |
| **T3** | 글로벌 시장조사기관 | Gartner, IDC, Statista, McKinsey, BCG, Forrester, Frost & Sullivan |
| **T4** | 학술 논문 | RISS, DBpia, KCI 등재지, IEEE/ACM/Nature 등 학술 DB, 국내외 학회 발표 자료 |
| **T5** | 기업 IR (타사) | DART 외 보조 IR 자료(투자자 대상 발표자료 등) |

### 사용 규칙

- **핵심 정량(시장규모·CAGR·점유율 Top-3·정책예산)** 인용은 **T1~T3에서**
- 보조 인용(질적 트렌드·기술 동향)은 T4 허용
- 자사 자료(자사 IR·자사 블로그) 자체인용 **금지** (순환 참조 광탈)
- `source_verification.md`에 **모든 인용의 Tier 컬럼** 의무화

### 차단 대상 (Tier 위계 제외 — 인용·리서치 모두 금지)

언론 기사·보도자료·블로그는 **정확성·이해관계 편향 리스크**로 사용하지 않는다. 정책 사실은 정부 사이트 원문에서, 시장 수치는 통계 채널에서 직접 확보한다.

#### 차단 도메인 리스트 (검색 결과 필터링 시 제외)

```
yna.co.kr, news.naver.com, news.daum.net, news.google.com,
hankyung.com, mk.co.kr, mt.co.kr, sedaily.com, etnews.com,
chosun.com, donga.com, joongang.co.kr, hani.co.kr, khan.co.kr,
kbs.co.kr, mbn.co.kr, sbs.co.kr, ytn.co.kr, news1.kr, newsis.com,
zdnet.co.kr, bloter.net, byline.network,
*.tistory.com, *.medium.com, brunch.co.kr, naver.com/blog/*, blog.naver.com,
prnewswire.com, businesswire.com (보도자료 배포)
```

추가 패턴:
- URL이 회사 자사 도메인과 일치 → 자사 자체인용 금지
- "보도자료", "press release", "PR Newswire" 표시 페이지 제외

## 2. 데이터 신선도 룰

기준일: 현재 연도(예: 2026). 발행연도 검증 후 초과 시 자동 **OUTDATED** 분류.

| 산업군 | 허용 발행연도 | 비고 |
|-------|-----------|------|
| ICT/AI/SaaS/반도체/바이오 | ≤ 2년 (2024 이후) | 변화 속도 빠름 |
| 일반 제조·서비스·소비재 | ≤ 3년 (2023 이후) | |
| 거시지표·정책·교육 | ≤ 5년 (2021 이후) | 안정적 영역 |

### 처리

- OUTDATED 판정 → 채택표에서 제외 후보로 표시(빨간 신선도 컬럼)
- 최신 자료 자동 재검색 트리거 (다음 연도 키워드 + `site:kosis.kr` 도메인 한정)
- 대체 불가 시 `[신선도 부족 — 인용 보류]` 태그

## 3. 교차검증 (Triangulation) 임계값

### 적용 대상

- 시장규모 (TAM, SAM, SOM)
- 시장 성장률 / CAGR
- 점유율 Top-3 경쟁사
- 정책 예산 / 지원금 규모

### 절차

1. 독립된 **2개+ 출처** 확보 의무
2. 수치 차이 계산: `(max − min) / min × 100%`
3. 판정:
   - **±10% 이내** → 채택, `채택값`은 두 출처의 보수값(min) 사용
   - **±10% 초과** → 보수적 값 + 두 출처 모두 명기, 차이 사유 1줄 기록
4. 1개 출처만 가능 → `[단일출처]` 태그 + Tier 1 강제 (T1 외 단일출처 인용 금지)

### source_verification.md 컬럼 표준

```
| # | 카테고리 | 인용값 | Tier | 출처1 (URL) | 출처2 (URL) | 일치도 | 채택값 | 신선도 | 자동검증 | 원문 스니펫(50자+) |
```

## 4. TAM/SAM/SOM 산출 산식 템플릿

### TAM (Top-down)
```
TAM = [산업 전체 매출 (KOSIS·KOTRA T1·T2 출처)] × [카테고리 비중]
```

### SAM (Top-down × 필터)
```
SAM = TAM × [지역 필터: 국내/특정 지역] × [세그먼트 필터: 고객군] × [규제 필터: 인허가 등]
```

### SOM (Bottom-up)
```
SOM = [타깃 고객 수] × [평균 단가(ASP)] × [3년차 침투율]
```

각 셀마다 (a) 산식, (b) 원자료 출처 URL/연도, (c) 가정의 보수성 근거를 함께 표기. reviewer 양식 게이트가 3개 셀 모두 채워졌는지 검증.

## 5. 광탈 패턴 자동 감지 (8종)

`source_verification.md` 작성 후 self-check를 수행하고 다음 패턴 발견 시 보강:

| # | 패턴 | 감지 로직 | 조치 |
|---|------|---------|------|
| 1 | 거대수치 단일 인용 | "전 세계", "글로벌" 등 키워드 + SOM 산식 누락 | Bottom-up 산식 보강 지시 |
| 2 | TAM=SAM=SOM | 3값이 동일 또는 ±5% 이내 | 필터링 산식 재작성 |
| 3 | 자사 자체인용 | URL 도메인 = 회사 도메인 | 인용 삭제 + 외부 출처 재검색 |
| 4 | 언론·블로그 인용 | URL 도메인이 §1 차단 리스트 일치 | 인용 삭제 + T1~T4 재검색 |
| 5 | 단일 출처 핵심 수치 | T1 아닌데 단일출처 | T1 보완 검색 후 교차검증 |
| 6 | 신선도 미달 | 발행연도가 §2 기준 초과 | 최신 자료 재검색 |
| 7 | 추정치를 사실처럼 | "약", "추정", "예상" 표현 + 단일출처 | "[추정]" 태그 명시화 또는 삭제 |
| 8 | 출처 3요소 누락 | 기관명·발행연도·URL 중 1개 이상 누락 | 인용 자격 박탈 |

## 6. KIPRIS 검색 5단계 절차

자의적 유사도 추정값을 폐지하고 객관 지표로 전환.

```
Step 1: 핵심 기술 1~2문장 요약 → 키워드 3~5개 추출
Step 2: KIPRIS 키워드 검색 → 청구항 일치도 가장 높은 모(母)특허 1건 식별
Step 3: 모특허의 IPC/CPC 분류 코드 추출 → 분류 검색으로 기술군 전수 조망
Step 4: 출원인 분석 → 산업 내 활동 주체(대기업/연구소/스타트업) 매핑
Step 5: 패밀리 특허(다국 출원) 확인 → 시장 확장성 근거
```

### kipris_search.md 출력 컬럼

```
| 순번 | 등록번호 | 특허명 | 출원인 | IPC/CPC | 청구항 키워드 매칭률 | 본 과제와의 차이점 |
```

- "유사도 65%" 같은 자의적 추정 폐지
- 매칭률은 키워드 N개 중 일치 개수 / N으로 계산 (예: 5개 중 3개 = 60%)

## 7. 다중 검색 전략 (토픽당 최소 4쿼리)

각 리서치 토픽 1건당 다음 4가지 쿼리를 병렬로 WebSearch:

```
1. 한국어 핵심 키워드        예: "AI SaaS 시장 규모 2025"
2. 한국어 동의어 변형        예: "인공지능 클라우드 서비스 시장"
3. 영어 키워드 (글로벌)      예: "Korea AI SaaS market size 2024"
4. 공식 도메인 한정          예: "site:kosis.kr 인공지능 시장"
```

결과 통합 → 차단 도메인 필터링 → Tier 정렬 → 신선도 필터 → 교차검증 후보 선정.

---

## 8. 공식 데이터 채널 우선순위

각 리서치 토픽별 1차 검색 채널:

| 토픽 | 1차 채널 (T1·T2) | 2차 채널 |
|------|----------------|---------|
| 시장규모/성장률 | KOSIS, KOTRA, 한국은행 ECOS | NIPA, KISDI |
| 경쟁사 매출 | DART 사업·반기보고서 | 글로벌 IR (T5) |
| 정책·예산 | 기획재정부 예산서, 중기부·과기정통부 정책자료 | 공공데이터포털 |
| 산업통계 | SGIS, KOSIS, 공공데이터포털 | 산업협회 백서 |
| 특허 | KIPRIS | 학술 DB |
| 글로벌 동향 | Gartner/IDC/Statista | OECD, World Bank |

도메인 한정 검색을 1차로 시도, 부족 시에만 일반 WebSearch로 확장. 일반 WebSearch 결과는 §1 차단 도메인을 필터링한 뒤 Tier 분류한다.
