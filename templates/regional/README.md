# 지역 R&D / 사업화 템플릿

## 유형 특성

- **대상**: 지자체 주관 R&D 또는 사업화 지원사업
- **핵심 전략**: 지역 기여 + 지역 파트너 + 지역 고용 극대화
- **평가 중점**: 지역 산업 생태계 연계, 지역 고용 창출, 지역 내 협력 네트워크
- **심사역 특성**: 지역 공무원 비율이 높을 수 있음 -- 전문용어 최소화
- **주의**: 기술 중심으로만 작성하면 "지역 기여가 안 보인다"는 평가

## 템플릿 채우기 가이드

1. 해당 공고의 작성서식(.hwpx)을 `hwpx_handler.py analyze`로 분석
2. 테이블 인덱스, 행/열, 빈셀(✎) 위치를 확인
3. `fill_template.json`의 cells 배열에 셀 좌표와 플레이스홀더 텍스트 추가
4. `sections_template.json`의 sections 배열에 header_keyword + pairs 추가
5. `auto_template_map.py`로 초안 자동 생성 가능

## 지역성 필수 포함 항목

- 기업 소재지 + 지역 기반 운영 현황
- 지역 내 고객사/파트너 확보 계획
- 지역 일자리 창출 기여 (채용 계획 연계)
- 지역 산업 생태계(특화 산업, 클러스터)와의 연계성
- 주관기관/지자체와의 시너지 포인트
