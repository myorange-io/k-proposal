# 예비창업패키지 / 초기창업패키지 템플릿

## 유형 특성

- **대상**: 예비창업자 또는 창업 3년 이내 기업
- **핵심 전략**: 아이디어 신선도 + 창업 의지 최대화. 기존 매출 강조 금지
- **분량**: 보통 10페이지 내외 (공고별 상이)
- **평가 중점**: 문제 정의의 구체성, 솔루션 참신성, 팀 실행 역량, 시장 검증 의지
- **매출**: 매출 0 허용. "아직 없다"가 감점이 아님
- **주의**: "이미 매출이 있다"를 강조하면 예비창업 취지에 맞지 않아 역효과

## 템플릿 채우기 가이드

1. 해당 공고의 작성서식(.hwpx)을 `hwpx_handler.py analyze`로 분석
2. 테이블 인덱스, 행/열, 빈셀(✎) 위치를 확인
3. `fill_template.json`의 cells 배열에 셀 좌표와 플레이스홀더 텍스트 추가
4. `sections_template.json`의 sections 배열에 header_keyword + pairs 추가
5. `auto_template_map.py`로 초안 자동 생성 가능

## 참고

- TIPS 템플릿(`templates/tips/`)의 구조를 참고하되, 평가항목과 양식 구조가 다르므로 그대로 복사하지 말 것
- 예비창업은 TIPS와 달리 운영사/선투자 개념이 없음
