# 창업도약패키지 템플릿

## 유형 특성

- **대상**: 창업 3~7년 이내 기업, 스케일업 단계
- **핵심 전략**: 시장 검증 증거(계약서, 매출) + 스케일업 시나리오 중심
- **평가 중점**: 기존 매출 성장 추이, 제품-시장 적합성(PMF) 검증, 스케일업 병목 해소 방안
- **매출**: 매출 실적이 있어야 유리. 성장률과 추이가 핵심
- **주의**: 초기 아이디어 단계처럼 작성하면 "도약이 아니라 시작 단계" 인상

## 템플릿 채우기 가이드

1. 해당 공고의 작성서식(.hwpx)을 `hwpx_handler.py analyze`로 분석
2. 테이블 인덱스, 행/열, 빈셀(✎) 위치를 확인
3. `fill_template.json`의 cells 배열에 셀 좌표와 플레이스홀더 텍스트 추가
4. `sections_template.json`의 sections 배열에 header_keyword + pairs 추가
5. `auto_template_map.py`로 초안 자동 생성 가능

## 참고

- 예비창업과 달리 **기존 실적(매출, 고객, 투자)**이 핵심 증거
- 스케일업 병목(인력, 기술, 시장 진입 장벽)을 명확히 정의하고 이 사업으로 해소하는 논리 필요
