# TIPS(팁스) 리서치 레퍼런스 2024-2026

AI가 TIPS 일반트랙 연구개발계획서 작성 시 자동 참조하는 레퍼런스 문서.

---

## 1. TIPS 프로그램 개요

**정식 명칭**: 민관공동창업자발굴육성(TIPS) — 창업성장기술개발사업

**핵심 구조**: 민간 운영사(VC/AC)가 선투자 + 추천 → 정부가 R&D 자금 매칭 지원. 정부지원사업이 아니라 **투자 프로그램의 연장선**.

**성장 사다리**:
```
Pre-TIPS → TIPS R&D(일반트랙) → Post-TIPS → 딥테크TIPS → 글로벌TIPS
```

## 2. 최근 3년 변천사

### 2024년
- 접수 방식: 수시 접수 (8차까지 진행)
- R&D 지원한도: 최대 5억원
- 선정 규모: 약 700개사

### 2025년
- 총 예산: 약 1,132.8억원
- 선정 규모: 약 820개사 (TIPS R&D + 사업화 + Pre/Post/글로벌 전체)
- 사다리형 구조 강화

### 2026년 대개편

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| R&D 지원금 | 최대 5억원 | **최대 8억원** (60% 증액) |
| 선정 규모 | 약 700개사 | **약 800개사** |
| 접수 방식 | 수시 | **분기별** (1, 2, 3분기 각 1회) |
| 선투자(수도권) | 1~2억원 | **2억원 이상** |
| 선투자(비수도권) | 1억원 | 1억원 유지 |
| 트랙 구조 | 일반/딥테크/글로벌 | **일반트랙 단일화** (이후 단계적 진행) |
| 가점 최대 | 3점 | **5점** |
| 비수도권 우대 | 가점 2점 | **선정 50% 우선 + 가점 최대 3점** |
| 일반 TIPS 예산 | 4,777억원 | **6,684억원** (39.9% 증) |
| 스케일업/글로벌 예산 | 1,567억원 | **3,671억원** (2.3배 증) |

## 3. 서류평가 기준 (100점 만점, 60점 미만 탈락)

### 4대 평가항목

**1. 문제인식**
- 시장의 실제 문제가 구체적인가
- 왜 지금 이것이 필요한가
- 기존 방식/솔루션의 한계는 무엇인가
- **핵심**: "불편하다"가 아니라 "최대 20%의 추가 비용 발생" 수준의 구체성

**2. 실현가능성**
- 2년 내 달성 가능한 구체적 로드맵이 있는가
- 6개월 단위 목표가 있는가
- TRL(기술성숙도)이 명시되어 있는가
- 기술적 난제와 해결방안이 있는가
- 현재 프로토타입/MVP 완성도는 어느 수준인가

**3. 성장전략**
- 비즈니스 모델이 명확한가 (누가 돈을 내는가)
- 초기 고객 확보 방안이 있는가
- 후속 투자 계획이 있는가
- 매출 산정에 객관적 근거(TAM/SAM/SOM)가 있는가
- 글로벌 확장 계획이 있는가

**4. 팀 구성**
- 2인 이상 팀인가
- CTO(기술책임자)/CSO(사업책임자) 역할 분담이 명확한가
- 팀원의 경력이 해결하려는 문제와 직접 관련 있는가
- 팀 시너지(기술+사업+마케팅)가 보이는가

### 가점 항목 (최대 5점)

| 항목 | 점수 | 취득 조건 |
|------|------|---------|
| 비수도권 본사 | 최대 3점 | 사업자등록증 주소 비수도권 |
| ESG 분야 사업 | 2점 | 임팩트 투자 + 관련 증빙 |
| Pre-TIPS 성공 판정 | 2점 | 기 수행 이력 |
| 벤처기업/이노비즈 인증 | 1점 | 유효한 인증서 보유 |
| 퇴직연금제도 도입 | 1점 | 제도 운영 증빙 |

## 4. 합격 기업 공통 패턴 (합격 시그널)

1. **기술 설명이 짧고 명확**: 핵심 기술을 3줄로 요약할 수 있음
2. **시장 문제가 구체적**: 수치와 그래프로 Pain Point 증명
3. **작은 실적이라도 있음**: 매출 100만원이라도, PoC 1건이라도
4. **팀 역할이 명확히 나뉨**: CTO/CSO 역할 분담, 각자의 전문 영역
5. **2년 안에 가능한 계획만 작성**: 비현실적 목표 없음
6. **IP로 기술 해자 증명**: 특허 출원/등록, 논문 게재
7. **신뢰도 높은 평가방법**: 공인시험기관 > 고객사 PoC > 자체평가
8. **후속 투자 시나리오 포함**: TIPS 이후 Post-TIPS, 시리즈A 계획
9. **경쟁사 분석이 구체적**: 기능 비교표, 가격 비교, 차별점 시각화

## 5. 탈락 기업 공통 패턴 (광탈 패턴)

### 언어 패턴 (쓰면 감점)
- "4차 산업혁명 시대에..." — 진부한 도입부
- "국내 유일" / "세계 최초" — 검증 불가
- "AI 기반 추천 시스템 개발" — 추상적 기술 설명
- "높은 정확도" / "빠른 처리속도" — 정량 수치 없음
- "혁신적인" / "차세대" / "게임체인저" — 추상적 수식어
- "~할 것으로 기대됩니다" — 막연한 미래 추정
- "본 사업은 ~을 목적으로 하며" — AI 작성 흔적

### 구조 패턴 (구성 오류)
- 6개월 단위 로드맵 없음 → 실현가능성 감점
- TRL 미기재 → 기술 수준 판단 불가
- 성능지표에 자체 평가만 제시
- 팀 2인 미만, CTO/CSO 역할 미분리
- 매출/PoC 0건인데 사업화 목표만 큼
- 경쟁사 비교 없이 차별성 주장
- 기술 설명이 전체의 50% 초과 → "학술 논문"
- 사업화 설명이 15% 미만 → "R&D 의지 부족"
- 예산 산출근거 "1식" → 감점 확정
- 비수도권인데 지역 가점 활용 안 함

## 6. 연구개발계획서 본문1 합격 작성법

### 본문 구성 순서 (별지 제3호 3-1)

1. **연구개발과제의 필요성** (T26)
   - Top-Down: 거시 트렌드 → 구체 문제
   - 수치+그래프: 시장 규모, 성장률, Pain Point 정량화
   - 시장의 빈틈: 기존 솔루션이 왜 이 문제를 못 풀었는가
   - 정책 연관성: 정부 정책과의 부합성

2. **해결방안** (T27)
   - 핵심 솔루션 한줄 요약
   - 기존 대비 차별점
   - 적용 시나리오 (사용자 관점)

3. **연구개발 목표 및 내용** (T28)
   - 시스템 아키텍처 다이어그램 (필수)
   - 핵심 알고리즘 플로우차트 (필수)
   - 모듈별 개발 내용
   - TRL 현재 → 목표

4. **결과물의 성능지표** (T29)
   - 정량 수치만 사용
   - 세계최고수준 대비 60-80% 목표가 권장 구간
   - 중간점검(1년차 60%) 마일스톤

5. **평가방법** (T31)
   - 공인시험기관 > 고객사 PoC > 자체평가 순서

6. **지식재산권 현황** (T33)
   - 보유특허 + 경쟁사 특허 모두 기재

7. **연구개발 현황** (T34)
   - 선행연구, 기술이전, 특허출원 등 준비 내용
   - MVP/프로토타입 현황, 확보 데이터

8. **추진일정** (T37)
   - 1차년도 + 2차년도, 12개월 단위
   - 6개월 단위 마일스톤 필수

9. **경쟁사 현황 + 시장현황 + 사업화 전략** (T38, T39, T40, T41)
   - 경쟁사 비교표 (기능/가격/성능)
   - TAM/SAM/SOM 기반 시장 분석
   - BM + 초기 고객 확보 전략
   - **후속 투자 시나리오 필수**: TIPS → Post-TIPS → 시리즈A

10. **기업소개** (T52)
    - 대표/CTO/CSO 역할 분담
    - 문제와 직결되는 경력
    - 팀 시너지

### 섹션별 분량 비율 (8억 규모)

- **기술 40%**: 필요성 + 해결방안 + 연구개발내용 + 성능지표 + 선행연구
- **사업화 30%**: 경쟁사 + 시장 + 사업화전략 + 글로벌 + 매출산정
- **팀/실적 30%**: 기업소개 + 추진체계 + 고용 + 운영사보육

8억 규모 R&D 기대치:
- 연구원 3-5명 풀타임 2년
- 논문 2편 이상
- 특허 3건 이상
- 쉬운 개발이면 "왜 8억이 필요한가?" 질문에 답할 수 없음

실제 급여가 고시 단가보다 낮으면 실제 지급액 기준.

### 비목별 한도

| 비목 | 한도/규칙 |
|------|---------|
| 인건비 | 계상률 100% 이내 |
| 연구수당 | 인건비의 20% 이내 |
| 위탁연구개발비 | 직접비의 40% 이내 |
| 외부전문기술활용비 | 직접비의 40% 이내 |
| 간접비 | 직접비의 10% 이내 (영리기관) |

### 산출근거 규칙

"1식" 단위 금지. 공수(인원 × 기간 × 단가) 기반 작성.

| 나쁜 예 (감점) | 좋은 예 |
|-------------|--------|
| AI 엔진 개발 1식 = 50,000천원 | AI 엔진 고도화 용역: 연구원 2인 × 6개월 × 월 2,901,312원 = 34,816천원 |
| 서버 구축 1식 = 10,000천원 | 클라우드 서버 임차: AWS m5.xlarge × 24개월 × 월 300천원 = 7,200천원 |

### 민간부담금 구성

- 정부출연금 최대 8억원 + 민간부담금(현금+현물)
- 현금 최소: 정부출연금의 10% 이상
- **현금 비율 60% 이상 권장** (사업 의지 바로미터)
- 운영사 선투자금이 민간부담금 현금에 포함

## 8. 최종평가 판정 기준 (역설계용)

### 판정 등급

| 등급 | 달성률 | 의미 |
|------|-------|-----|
| 우수 | 90% 이상 | 목표 초과 달성 |
| 보통 | 60-89% | 목표 대부분 달성 |
| 미흡 | 60% 미만 | 미달성 → **환수 위험** |

### 성능지표 목표 설정 밸런스

| 세계최고 대비 구간 | 선정 가능성 | 최종평가 위험 | 판정 |
|-----------------|-----------|------------|------|
| 50% 미만 | 낮음 | 안전 | 상향 필요 |
| 60-80% | 적정 | 안전 | **권장 구간** |
| 80-90% | 높음 | 보통 | 근거 충분 시 허용 |
| 90-100% | 매우 높음 | 높음 | 경고 |
| 100% 초과 | 비현실적 | 매우 높음 | 금지 |

- 달성 가능성 80% 이상을 내부 기준으로 설정
- 중간점검: 각 지표에 "1년차 60% 달성" 마일스톤

## 9. 운영사 추천 흐름 + 선투자 조건

### 진행 흐름

1. 운영사 미팅 (IR Deck 제출)
2. 투자심사 (내부 검토)
3. 투자 확약서 발급
4. 운영사 추천 등록
5. IRIS 시스템 신청
6. 정부 대면평가

### 선투자 조건

- 수도권: 2억원 이상
- 비수도권: 1억원 이상
- 지분 조건: 운영사 30% 이하, 창업팀 60% 이상

### 운영사 유형별 특성

| 유형 | 예시 | 특징 |
|------|-----|------|
| 대형 VC | 본엔젤스, 블루포인트 | IT/SaaS, 딥테크 전문 |
| 초기 전문 VC | 매쉬업벤처스, 소풍벤처스 | 시드~시리즈A |
| 창업기획자(AC) | 퓨처플레이, 스파크랩 | 로봇/AI, 다분야 |
| 대학/연구소 연계 | KAIST 보유 운영사 등 | 기술이전 연계 |

## 11. 후속 투자 사다리

```
현재 ─── TIPS R&D(2년) ──── Post-TIPS ────── 시리즈A ────── 스케일업
선투자2억  R&D 8억(정부)    추가 7억(정부)   민간투자      매출기반성장
           기술 완성         사업화 가속       시장 확대      글로벌 진출
```

### 단계별 목표 (예시)

| 단계 | TRL | 특허 | 매출 | 투자 | 기업가치 |
|------|-----|------|------|------|---------|
| TIPS 시작 | 4-5 | 1-2건 출원 | PoC 수준 | 선투자 2억 | 10-20억 |
| TIPS 완료 | 7-8 | 3건+ 등록 | X억 | - | 50-100억 |
| Post-TIPS 완료 | 8-9 | 5건+ | Y억 | 후속 Z억 | 100-200억 |
| 시리즈A | 9 | 글로벌 출원 | 매출 기반 | 시리즈A | 200억+ |
